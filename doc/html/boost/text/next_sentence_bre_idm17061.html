Function template next_sentence_break

boost::text::next_sentence_break

Synopsis

// In header: <boost/text/sentence_break.hpp>


template<typename CPRange, typename CPIter> 
  unspecified next_sentence_break(CPRange & range, CPIter it);

Description

Finds the next sentence break after it. This will be the first code point after the current sentence, or range.end() if no next sentence exists.

This function only participates in overload resolution if CPRange models the CPRange concept.

Requires:

it is at the beginning of a sentence.
