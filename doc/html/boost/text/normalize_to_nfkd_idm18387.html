Function normalize_to_nfkd

boost::text::normalize_to_nfkd

Synopsis

// In header: <boost/text/normalize_string.hpp>


void normalize_to_nfkd(string & s);

Description

Puts the contents of s in Unicode normalization form NFKD. Normalization is not performed if s passes a normalization quick-check.
