Function checked_encoding

boost::text::checked_encoding

Synopsis

// In header: <boost/text/rope.hpp>


rope const & checked_encoding(rope const & r);

Description

Forwards r when it is entirely UTF-8 encoded; throws otherwise.

Throws:

 std::invalid_argument when r is not UTF-8 encoded.
