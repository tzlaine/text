Function template find_if_backward

boost::text::find_if_backward

Synopsis

// In header: <boost/text/algorithm.hpp>


template<typename BidiIter, typename Pred> 
  BidiIter find_if_backward(BidiIter first, BidiIter last, Pred p);

Description

Analogue of std::find() that finds the last value v in [first, last) for which p(v) is true.
