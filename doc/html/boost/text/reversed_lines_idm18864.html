Function template reversed_lines

boost::text::reversed_lines

Synopsis

// In header: <boost/text/line_break.hpp>


template<typename GraphemeRange> 
  unspecified reversed_lines(GraphemeRange const & range);

Description

Returns a lazy range of the grapheme ranges delimiting lines (using hard line breaks) in range, in reverse.

This function only participates in overload resolution if GraphemeRange models the GraphemeRange concept.
