Function high_surrogate

boost::text::high_surrogate

Synopsis

// In header: <boost/text/transcode_iterator.hpp>


constexpr bool high_surrogate(uint32_t c);

Description

Returns true if c is a Unicode high surrogate, or false otherwise.

This function is constexpr in C++14 and later.
