Function template operator+

boost::text::operator+

Synopsis

// In header: <boost/text/text.hpp>


template<typename CharRange> 
  text operator+(CharRange const & r, text const & t);

Description

Creates a new text object that is the concatenation of r and t.

This function only participates in overload resolution if CharRange models the Char_range concept.

Throws:

 std::invalid_argument if the ends of the range are not valid UTF-8.
