Function continuation

boost::text::utf8::continuation

Synopsis

// In header: <boost/text/utf8.hpp>


constexpr bool 
continuation(unsigned char c, unsigned char lo = 0x80, 
             unsigned char hi = 0xbf);

Description

Returns true if c is a UTF-8 continuation code unit, and false otherwise. If optional parameters lo and hi are given, the code unit must also lie in the range [lo, hi].
