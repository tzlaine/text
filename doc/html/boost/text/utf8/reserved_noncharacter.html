Function reserved_noncharacter

boost::text::utf8::reserved_noncharacter

Synopsis

// In header: <boost/text/utf8.hpp>


constexpr bool reserved_noncharacter(uint32_t c);

Description

Returns true if c is a Unicode reserved noncharacter, or false otherwise.

This function is constexpr in C++14 and later.

See Also:

Unicode 9, 3.4/D14
