Function template to_lower

boost::text::to_lower

Synopsis

// In header: <boost/text/case_mapping.hpp>


template<typename CPIter, typename Sentinel, typename OutIter> 
  OutIter to_lower(CPIter first, CPIter it, Sentinel last, OutIter out, 
                   case_language lang = case_language::other);

Description

Writes the code point sequence comprising the lower-case form of [first, last) to out, using language-specific handling as indicated by lang. The normalization of the result is undefined.
