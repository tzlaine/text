Function template sentence

boost::text::sentence

Synopsis

// In header: <boost/text/sentence_break.hpp>


template<typename GraphemeRange, typename GraphemeIter> 
  unspecified sentence(GraphemeRange const & range, GraphemeIter it);

Description

Returns grapheme range delimiting the bounds of the sentence that it lies within, as a grapheme_view.

This function only participates in overload resolution if GraphemeRange models the GraphemeRange concept.
