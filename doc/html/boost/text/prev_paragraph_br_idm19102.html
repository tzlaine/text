Function template prev_paragraph_break

boost::text::prev_paragraph_break

Synopsis

// In header: <boost/text/paragraph_break.hpp>


template<typename CPIter, typename Sentinel> 
  CPIter prev_paragraph_break(CPIter first, CPIter it, Sentinel last);

Description

Finds the nearest paragraph break at or before before it. If it == first, that is returned. Otherwise, the first code point of the paragraph that it is within is returned (even if it is already at the first code point of a paragraph).

This function only participates in overload resolution if CPIter models the CPIter concept.
