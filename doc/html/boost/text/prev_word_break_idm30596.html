Function template prev_word_break

boost::text::prev_word_break

Synopsis

// In header: <boost/text/word_break.hpp>


template<typename CPRange, typename CPIter, 
         typename WordPropFunc = word_prop_callable, 
         typename CPWordBreakFunc = detail::undefined> 
  unspecified prev_word_break(CPRange & range, CPIter it, 
                              WordPropFunc word_prop = WordPropFunc{}, 
                              CPWordBreakFunc cp_break = CPWordBreakFunc{});

Description

Finds the nearest word break at or before before it. If it == range.begin(), that is returned. Otherwise, the first code point of the word that it is within is returned (even if it is already at the first code point of a word).

This function only participates in overload resolution if CPRange models the CPRange concept and WordPropFunc models the WordPropFunc concept.
