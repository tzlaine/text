Function template ends_encoded

boost::text::v1::ends_encoded

Synopsis

// In header: <boost/text/transcode_iterator.hpp>


template<typename Iter> unspecified ends_encoded(Iter first, Iter last);

Description

Returns true if [first, last) is empty or the final code units in [first, last) form a valid Unicode code point, or false otherwise.
