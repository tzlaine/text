Function template from_utf16_front_inserter

boost::text::v1::from_utf16_front_inserter

Synopsis

// In header: <boost/text/transcode_iterator.hpp>


template<typename Cont> auto from_utf16_front_inserter(Cont & c);

Description

Returns a front-inserting iterator that transcodes from UTF-16 to UTF-8, UTF-16, or UTF-32. Which UTF the iterator transcodes to depends on sizeof(Cont::value_type): 1 implies UTF-8; 2 implies UTF-16; and any other size implies UTF-32.
