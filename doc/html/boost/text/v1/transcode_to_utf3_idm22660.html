Function template transcode_to_utf32

boost::text::v1::transcode_to_utf32

Synopsis

// In header: <boost/text/transcode_algorithm.hpp>


template<typename Iter, typename Sentinel, typename OutIter> 
  OutIter transcode_to_utf32(Iter first, Sentinel last, OutIter out);

Description

Copies the code points in the range [first, last) to out, changing the encoding to UTF-32.
