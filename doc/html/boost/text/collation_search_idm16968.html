Function template collation_search

boost::text::collation_search

Synopsis

// In header: <boost/text/collation_search.hpp>


template<typename CPIter, typename Sentinel, typename Searcher> 
  cp_range< CPIter > 
  collation_search(CPIter first, Sentinel last, Searcher const & searcher);

Description

Returns the code point subrange within [first, last) in which searcher finds its pattern. If the pattern is not found, the resulting range will be empty.

This function only participates in overload resolution if searcher(first, last) is well formed.
