Function template prev_allowed_line_break

boost::text::prev_allowed_line_break

Synopsis

// In header: <boost/text/line_break.hpp>


template<typename GraphemeRange, typename GraphemeIter> 
  unspecified prev_allowed_line_break(GraphemeRange const & range, 
                                      GraphemeIter it);

Description

Returns a grapheme_iterator to the nearest line break opportunity at or before before it. If it == range.begin(), that is returned. Otherwise, the first grapheme of the line that it is within is returned (even if it is already at the first grapheme of a line).

This function only participates in overload resolution if GraphemeRange models the GraphemeRange concept.
