Function utf16_code_units

boost::text::utf16_code_units

Synopsis

// In header: <boost/text/transcode_iterator.hpp>


constexpr int utf16_code_units(uint16_t first_unit);

Description

Given the first (and possibly only) code unit of a UTF-16-encoded code point, returns the number of code units occupied by that code point (in the range [1, 2]). Returns a negative value if first_unit is not a valid initial UTF-16 code unit.
