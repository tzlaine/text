Struct template line_break_result

boost::text::line_break_result

Synopsis

// In header: <boost/text/line_break.hpp>

template<typename CPIter> 
struct line_break_result {

  // public data members
  CPIter iter;
  bool hard_break;
};

Description

The result type for line break algorithms that return an iterator, and which may return an iterator to either a hard (i.e. mandatory) or allowed line break. A hard break occurs only after a code point with the line break property BK, CR, LF, or NL (but not within a CR/LF pair).
